=head1 NAME

FLTK::ReturnButton - 12:50, press Return

=head1 Description

Same as a normal L<button|FLTK::Button> except the
L<C<shortcut( )>|FLTK::Widget/"shortcut"> is preset to C<ReturnKey> and
C<KeypadEnter>, and a glyph is drawn to indicate this.
