=head1 NAME

FLTK::FlatBox - Box drawing code for the Fast Light Tool Kit (FLTK)

=head1 Description

Draws a rectangle filled with L<C<getbgcolor()>|/"getbgcolor">. This is a
useful base class for some box types.
